This folder is where you should store images for your blog posts.

To keep things organized, create a new folder in here for each post, named exactly like the post's `slug`.

For example:
If your post slug is `my-experience-with-openclaw`, create a folder here called `my-experience-with-openclaw` and put your images inside it.

Then, in your Markdown file, you can reference the image like this:
`![My Image](/images/posts/my-experience-with-openclaw/screenshot.png)`
